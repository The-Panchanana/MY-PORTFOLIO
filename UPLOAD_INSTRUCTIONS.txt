GITHUB UPLOAD INSTRUCTIONS
==========================

Since Git is not installed, follow these steps to upload your project:

METHOD 1: GitHub Web Interface (Easiest)
----------------------------------------
1. Go to: https://github.com/The-Panchanana/MY-PORTFOLIO
2. Click "uploading an existing file" or "Add file" > "Upload files"
3. Drag and drop ALL files from this folder EXCEPT:
   - node_modules folder (if exists)
   - .git folder (if exists)
4. Write commit message: "Initial portfolio upload"
5. Click "Commit changes"

METHOD 2: Install Git and Use Commands
-------------------------------------
1. Download Git: https://git-scm.com/download/win
2. Install with default settings
3. Open Command Prompt in this folder
4. Run these commands:

git init
git add .
git commit -m "Initial portfolio upload"
git branch -M main
git remote add origin https://github.com/The-Panchanana/MY-PORTFOLIO.git
git push -u origin main

FILES TO UPLOAD:
===============
✓ src/ (entire folder)
✓ public/ (entire folder)
✓ package.json
✓ README.md
✓ .gitignore
✓ UPLOAD_INSTRUCTIONS.txt

DO NOT UPLOAD:
=============
✗ node_modules/ (too large, will be installed via npm)
✗ .git/ (if exists)

Your project is ready for GitHub!
