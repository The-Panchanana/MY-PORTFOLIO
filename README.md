# MY-PORTFOLIO

A modern, responsive portfolio website built with React.js showcasing my skills as a Software Engineer specializing in SAP ABAP Development and Full Stack Web Development.

## Features

- **Responsive Design**: Works on all devices
- **Interactive UI**: Smooth animations and hover effects
- **Contact Form**: Functional email integration
- **Skills Showcase**: Toggle-able skill sections
- **Project Gallery**: Detailed project descriptions
- **Modern Styling**: Sky blue, white, and black color scheme

## Technologies Used

- React.js
- HTML5 & CSS3
- JavaScript (ES6+)
- Responsive Design
- Google Fonts

## Getting Started

1. Clone the repository
```bash
git clone https://github.com/The-Panchanana/MY-PORTFOLIO.git
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
